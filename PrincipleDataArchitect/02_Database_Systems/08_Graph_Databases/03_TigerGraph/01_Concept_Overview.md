# TigerGraph — Concept Overview

> The high-performance analytics-focused graph database for deep-link queries.

## Why TigerGraph Is Different

TigerGraph was built from scratch for **graph analytics** (not just traversal). Its GSQL query language supports graph algorithms natively, and its massively parallel processing (MPP) engine can traverse 100B+ edges in seconds.

| Feature | TigerGraph | Neo4j | Neptune |
|---|---|---|---|
| **Focus** | Graph analytics | OLTP graph | Managed graph |
| **Query Language** | GSQL (Turing-complete) | Cypher | Gremlin/SPARQL |
| **Processing** | MPP (distributed) | Single-threaded traversal | Managed scaling |
| **Scale** | 100B+ edges | Billions (Enterprise) | Billions (managed) |
| **Best For** | Fraud detection, ML features | Social networks, recommendations | AWS-native |
